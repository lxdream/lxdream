## Lxdream

Lxdream is an emulator that is still in heavy development (and many features are buggy or unimplemented), it is capable of running most demos and some games.

## Quick start

Several quick start options are available:

- [Download the latest release](https://github.com/lxdream/lxdream/archive/master.zip)
- Clone the repo: `git clone https://github.com/lxdream/lxdream.git`

- Create a fork: Click the `Fork` button on the top right.
